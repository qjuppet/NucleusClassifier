# Nuclei Classifier

Nuclei Classifier is an ImageJ/Fiji plugin that aim to classify nuclei based on various features related to the nuclei and their neighbours.
The detection is done with the [StarDist plugin](https://github.com/mpicbg-csbd/stardist).
